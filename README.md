# uvm_easy_mam

A memory allocation Manager  based uvm_mem_mam，with good usability.

uvm_easy_mam是基于uvm_mem_mam开发的memory地址空间管理组件，具有良好的使用接口，在SOC总线类验证中可以广泛使用。

## 接口功能

1. 支持函数接口添加多块memory规格信息

2. 支持函数接口形式的地址自动分配

3. 支持在指定memory中进行地址分配

4. 支持函数接口形式的地址空间释放

更多的功能待开发后合入。
